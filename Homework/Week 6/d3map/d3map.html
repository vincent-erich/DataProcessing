Population (in millions) by country in 2014

The map above show the population in millions by country in 2014. The data is from The World Bank. Hover over the map to see the population per country.

I have made a number of design choices while implementing this chloropleth map. First, I have drawn a small, black border around each country. I have done this, because the fill color for a population < five million interferes with the background color of the map; the black border provides contrast. Second, country names and raw data values (i.e., population) appear on mouse over; this information is shown as a tooltip. The tooltip appears directly under the mouse so that the information is clearly visible. Furthermore, a country is highlighted on mouse over so that it is clear which country is currently selected.
